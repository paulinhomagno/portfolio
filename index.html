Projetos em destaque

Desafio de previsão de falhas em máquinas

Desenvolvimento de um modelo de predição capaz de prever falhas em máquinas através da análise e modelagem da base de dados fornecida por uma empresa como parte de um processo seletivo. O modelo teve uma performance de 84% em identificar se há falha e o tipo de falha, e de todas as máquinas com falhas o modelo consegue prever 89%. Este projeto recebeu a nota de 4.8 de 5 pelos avaliadores.

Ferramentas utilizadas:

  • Git e Github.
  • Python (Pandas, Seaborn, Matplotlib, Sklearn, Scipy, Pickle).
  • Jupyter Notebook.
  • Técnicas de balancamento de dados: Near Miss, SMOTE.
  • Modelos Preditivos: Random Forest, LGBM, SVM.
